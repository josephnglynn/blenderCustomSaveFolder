## Usage
Open the Preferences ‣ Add-ons ‣ Install… and select the downloaded python file.

voila, a new rendering tab should appear and hey presto your renders will now save to %userprofile%/Documents/Blender Render Outputs/{project name}/{project Name + version}.{your set file type}
